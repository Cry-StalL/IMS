# IMS
这里是？

这里是一个简易学生管理系统，用于软件工程实训课大作业

此项目虚构了一个112589大学: 双鸭山大学，实现了其大学的基础教务管理后台系统，包括:

> 1. 学生、教师信息的增删改查
> 2. 选课、评教系统
> 3. 学生绩点、成绩、课程表的动态展示
> 4. 考试安排

实现工具:

> 1. SpringBoot 3.2.0
> 2. Java 21
> 3. Vue 3.2.13
> 4. ElementPlus 2.4.3
